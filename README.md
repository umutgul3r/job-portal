# job-portal

## Run Locally

### 1. Clone repo

```
$ git clone https://github.com/umutgul3r/job-portal.git
$ cd 44-Store
```

### 2. Install MongoDB

Download it from here: https://docs.mongodb.com/manual/administration/install-community/

### 3. Run Backend

```
$ cd api
$ npm install
$ npm run dev
```

### 4. Run Frontend

```
# open new terminal
$ cd frontend
$ npm install
$ npm start
```
